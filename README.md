I build practical AI systems at the intersection of retrieval, language models, and healthcare. My recent work includes Radiant, a rare-disease RAG chat agent described in Springer AIME 2025, and Tiny Transformer, a compact GPT-style model built from scratch in PyTorch with BPE tokenization, attention visualization, tests, and a Gradio playground.
